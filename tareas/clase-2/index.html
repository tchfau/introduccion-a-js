Clase 2: Tarea

Ejercicio 1:

Tenemos 2 funciones de la tarea anterior -- 'sumar' y 'restar'.
Vamos a decirle a la máquina qué función tiene que correr, dependiendo del operador artmético (+, -, /, *, etc.)
Si el operador es '+', vamos a usar la función para sumar.
Si no, vamos a usar la función para restar.

  1. Crear una variable llamada 'operador' y asignarle el valor '+'.
  2. Crear 2 variables que contengan 2 números cualquiera.
  3. Crear una condición if/else basada en el operador que tenemos.
  4. Si tenemos un operador igual a '+', llamamos la función 'sumar' con nuestros números (las variables del paso 2).
  5. Si no, tenemos que llamar la función 'restar', con nuestros números (las variables del paso 2).
  6. No se olviden de hacer un console.log para ver el resultado!

Ejercicio 2:

Cambiar el operador a '-', y fijarse que llame a la función 'restar' en vez de a la de 'sumar'.
